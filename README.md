# EcoVision-BOT
EcoVision Bot adalah bot Discord berbasis AI dan visi komputer yang bisa mengenali jenis sampah dari gambar yang dikirim oleh pengguna â€” seperti plastik, kertas, logam, organik, dan lainnya. Bot ini membantu pengguna belajar memilah sampah dengan benar, sambil membuat edukasi lingkungan jadi seru langsung di Discord.


## Inilah jenis proyek saya:
Ide saya adalah membuat bot Discord berbasis kecerdasan buatan (AI) yang dapat mengenali jenis sampah dari gambar yang dikirim oleh pengguna.
Bot ini menggabungkan teknologi visi komputer (computer vision) dengan platform Discord agar pengguna bisa belajar memilah sampah secara interaktif dan menyenangkan.
